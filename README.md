Unbiased AI Decision System

Overview

The Unbiased AI Decision System is a web-based application designed to detect, explain, and reduce bias in decision-making datasets. It provides an end-to-end pipeline that helps users identify unfair patterns, understand their impact, and generate a fairness-improved dataset.

Problem Statement

AI systems can inherit hidden biases from data, leading to unfair decisions (e.g., in hiring, loans, approvals). Detecting and correcting this bias is often complex and requires technical expertise.

Solution

Our system simplifies this process by offering:
Bias detection using fairness metrics
AI-powered explanations
Bias mitigation techniques
Downloadable corrected datasets

Tech Stack

React (Vite), Tailwind CSS, PapaParse, Recharts, Gemini API, Render

How It Works

Upload CSV → Select columns → Detect bias → Get AI insights → Reduce bias → Download improved data
